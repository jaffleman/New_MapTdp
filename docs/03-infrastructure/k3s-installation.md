## **Installation du cluster K3s**

### Points principaux

*   cluster léger, idéal pour small-to-medium workloads
*   installation rapide (systemd, binaire unique)
*   Traefik installé par défaut
*   support natif des Ingress

### Conseils

*   séparer control plane et worker si possible
*   activer une VIP si cluster HA
*   surveiller la santé des nodes
