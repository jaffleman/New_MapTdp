## **Backend Node.js / Express**

### Rôle

Le backend expose une API REST permettant :

*   de consulter les colonnes
*   de retrouver les réglettes
*   de renvoyer les informations techniques nécessaires en salle de dégroupage

### Fonctionnalités principales

*   récupération d’entrées depuis PostgreSQL
*   validation des paramètres
*   gestion des erreurs
*   logs applicatifs

### Dépendances

*   Express
*   PostgreSQL driver
*   middlewares standards (CORS, JSON parser, helmet…)
