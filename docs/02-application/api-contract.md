## **Contrat API**

### Objectifs

*   fournir un accès structuré aux données MapTdp
*   exposer les informations nécessaires à l’interface utilisateur
*   garantir une stabilité dans les réponses

### Structure des réponses

Les endpoints retournent toujours :

*   un statut
*   des données structurées
*   un éventuel message d’erreur
